Model Automation

Project Description I observed that a company's product was modular in nature, and started a business case as I saw that it had potential to be automated.
Business Benefits Decrease in drafting and 3D modelling time by using this product configurator tool
Potential for customers to use the product configurator tool in the future
Toolset C#, DriveWorks, SolidWorks
Methodology 1) Understand the product being built and the possible combinations of parts needed to build the system
2) Plan for all combinations of parts and create a process flow map for coding
3) Code the product configurator
Achievements Reduced Engineering department's drafting time by 50% when compared to the time to create a drawing manually
